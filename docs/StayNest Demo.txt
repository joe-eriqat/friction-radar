--COMPANY DESCRIPTION
StayNest is a vacation-rental marketplace for booking short-term stays in homes, cabins, apartments, guest suites, and unique local properties. Its main features include verified listings, upfront pricing breakdowns, map-based search, host messaging, digital check-in, house manuals, group wishlists, voting, split payments, damage-deposit handling, host calendar sync, smart pricing, and post-stay reviews.
